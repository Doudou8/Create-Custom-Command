# Create-Custom-Command

> Un CLI pour créer des CLI  
> **Important:** Ce CLI n'a été testé que sur Ubuntu 18.04 (LTS) avec ZSH

## Installation

```bash
sudo npm i -g create-custom-command
```

## Utilisation

> Pour créer une commande

```bash
mk-cmd
```

> Pour supprimer une commande

```bash
rm-cmd
```
